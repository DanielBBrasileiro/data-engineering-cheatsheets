# Data Engineering Cheatsheets

Practical, concise and printable cheatsheets focused on **Data Engineering**, 
covering **interviews**, **system design**, and **real-world pipelines**.

This repository follows the **80/20 principle**: what actually matters for 
production and international interviews.

### Docker

### Linux
- File system

- Troubleshooting

### Airflow

### Apache Spark

- Performance tuning

### SQL for Data Engineers

- Query optimization
- Analytics patterns
